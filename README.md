# WebMining

This is my graduation project, which is demonstration of the Web Usage Mining technologies.


Alogrithm used in this project:
  - Association Rules (apriori)
  - Clustring (DBSCAN)
  - Classification (K-NN)
  - MarkovChain
  - Statistic
